# 🏥 Medical Assistance App

> **Your Health, Immediate Priority** — Connect with top doctors, find nearby hospitals, and get emergency help within seconds.

---

## 📖 About the Project

A web-based medical assistance platform designed to make healthcare accessible and fast. From locating the nearest hospital to managing your health records — everything in one place, with an SOS emergency feature for critical situations.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🏥 **Find Hospital** | Locate nearest medical centers instantly |
| 📅 **Appointments** | Schedule visits with specialists |
| 📋 **Health Records** | Access and manage your medical history |
| 🧪 **Lab Reports** | View diagnostic results anytime |
| 🕐 **Patient History** | Track your complete medical timeline |
| 🆘 **SOS Emergency** | One-tap emergency call for critical situations |

---

## 🛠️ Tech Stack

- **HTML** — Structure & layout
- **CSS** — Styling & responsive design
- **JavaScript** — Interactivity & dynamic features

---

## 🚀 Getting Started

### Prerequisites
Just a modern web browser — no installation needed!

### Run Locally

```bash
# Clone the repository
git clone https://github.com/karishma-spec/medical-assistance-app.git

# Navigate to the project folder
cd medical-assistance-app

# Open in browser
open index.html
```
